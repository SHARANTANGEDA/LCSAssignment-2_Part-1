<creating a character stack 's'

<creating a stringstream called post













iterating through the character array 'arr[]'

if 'i'th element of 'arr[]' is not an operator symbol then append 'arr[i]' to post

if stack 's' is empty and 'i'th element of 'arr[]' is an operator push 'arr[i]' onto top of stack s










if stack 's' is not empty and 'i'th element of 'arr[]' is an operator we proceed based on type of operator 

if 'arr[i]' is '('

push 'arr[i]' onto top of stack s

else if 'arr[i]' is ')'

traverse down the stack s by appending top most element to 'post' and then 'pop'ing stack until we reach a '('

then remove '(' from top of stack

if arr[i] is any other operator

we traverse down the stack s by appending top most element of stack to 'post' and then 'pop'ing stack 's' until precedence of top of stack is less than precedence of 'arr[i]' (function for precedence is is defined later )

then push arr[i] to top of stack

After completion of above loop append the topmost element of stack 's' to 'post' until stack is empty

<printing the required post fix value
